Synlighetsmodifikatorer:

Synlighetsmodifikatorer i programmering er måter å deklarere hvilke variabler/metoder som 
skal være tilgjengelige og hvilke som skal være private. Dette kan gjøres i JAVA med 3 
forskjellige synlighetsmodifikatorer:

-	Public (tilgjengelig for alle klasser i prosjektet)
-	Private (kun tilgjengelig i klassen)
-	Ingen modifikator (kun tilgjengelig innad i pakken)
- 	Protected (tilgjengelig for klasser i pakken, men også subclasses i andre pakker)
